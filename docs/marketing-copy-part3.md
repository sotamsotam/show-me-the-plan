# 4. 서브 — `/elementary` (초등)

> **테마**: **공부 습관의 첫걸음**

---

## 4-1. Hero

**헤드라인**
> 공부는 엄마를 위해서가 아니라,  
> **'선생님과의 약속(숙제)'** 부터.

**서브카피**
> 초등 공부의 시작은 거창한 입시 전략이 아닙니다.  
> **숙제·예습·독서**를 약속처럼 지키는 습관, 그리고 **부모와 함께 보는 루틴**입니다.

**CTA**: **초등학생으로 시작하기** | 학부모용 → `/for-parents`

---

## 4-2. 이 학령의 고민

| # | 고민 |
|---|------|
| 1 | 플래너를 써도 며칠이면 그만둔다 |
| 2 | 학교 숙제·학원·독서 시간을 한눈에 정리하기 어렵다 |
| 3 | "공부했니?"가 매일 잔소리로 느껴진다 |
| 4 | 방학이 되면 공부 리듬이 완전히 깨진다 |

---

## 4-3. Show Me The Plan 접근

**본문**
> 초등학생에게는 **복잡한 통계보다, 오늘 할 일을 끝내는 경험**이 중요합니다.  
> NEIS 시간표에 맞춰 **학교 수업 → 자습 → 학원** 시간을 시각적으로 배치하고,  
> **방학 주차별 계획**으로 무너지지 않는 생활 리듬을 만듭니다.

---

## 4-4. 특화 기능

| 기능 | 설명 |
|------|------|
| **학교·자습·학원 시간 시각화** | NEIS 시간표 + 스터디 플랜을 캘린더에서 함께 확인 |
| **방학 루틴 유지** | 여름·겨울 방학 주차별 계획으로 습관 이어가기 |
| **분량 중심 TODO** | "독서 30분", "수학 익힘 5쪽"처럼 끝낼 수 있는 미션 |
| **매니저(부모) 연동** | 계획·실행을 함께 보며 **칭찬과 격려**의 도구로 활용 |

---

## 4-5. 전략 (학부모 + 아이)

**섹션 제목**
> 부모와 아이가 함께 쓰는 공부 도구

- 아이: 오늘 TODO 확인 → 완료 체크
- 부모(매니저): 이번 주 달성률 확인 → "3개 중 2개 했네, 내일 하나 더!"
- **잔소리 대신, 구체적 칭찬**

---

> 우리 아이의 **첫 공부 습관**, 함께 시작해 보세요.

**CTA**: **초등학생으로 시작하기**

# 5. 서브 — `/middle` (중등)

> **테마**: **늘어나는 과목, 전략적인 내신 대비**

---

## 5-1. Hero

**헤드라인**
> 벼락치기는 이제 안녕!  
> **시험 2~4주 전**부터 시작하는 역산형 주차 계획.

**서브카피**

> Show Me The Plan으로 **시험 D-day부터 거꾸로** 세운 미션을, 매일 TODO로 실행하세요.

**CTA**: **중학생으로 시작하기**

---

## 5-2. 이 학령의 고민

| # | 고민 |
|---|------|
| 1 | 과목이 늘어나 뭐부터 해야 할지 모르겠다 |
| 2 | 시험 2~3주 전부터 벼락치기 패턴 반복 |
| 3 | 학교·학원·자습 시간 합치면 계획표가 너무 복잡 |
| 4 | 시험 끝나면 리듬이 끊기고 다음 시험까지 방치 |

---

## 5-3. Show Me The Plan 접근

**본문**
> 중학교는 **의무감이 생기기 시작하는 시기**입니다.  
> 스스로 계획하고, 지키고, **성취감을 느끼는 경험**이 중요합니다.  
> 시험 준비 / 평소 / 방학 **3모드**로 상황에 맞게 계획을 전환하세요.

---

## 5-4. 특화 기능

| 기능 | 설명 |
|------|------|
| **역산형 시험 대비 계획** | 시험 기간 설정 → 주차별·과목별 분량 계획 |
| **3모드 전환** | 시험 준비 / 평소 / 방학 — 유연한 계획 전환 |
| **과목·교재 태그** | 교과서·참고서·프린트 등 출처별 과목 관리 |
| **지난 시험 복기** | 공부현황·통계로 이번 시험 기간 실행 분석 → 다음 계획 수정 |
| **NEIS + 학원 스케줄** | 학교 시간표와 자습·학원 일정 통합 |

> **카피 표현 참고**: "시험 출처 분석"은 과목·교재 태그 + 실행 통계 기반 복기로 표현. 전용 분석 기능은 부록 B 참고.
